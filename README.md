# 🚀 Task Manager Web App

A modern and efficient **Task Management Web Application** built using **Vanilla JavaScript**, designed to help users manage daily tasks with ease. The application supports task creation, completion tracking, filtering, and persistent storage using the browser's LocalStorage.

---

## 📌 Features

* ✅ Add new tasks dynamically
* ✅ Mark tasks as completed / pending
* ✅ Delete tasks instantly
* ✅ Filter tasks (All / Completed / Pending)
* ✅ Persistent data using LocalStorage
* ✅ Clean and minimal UI
* ✅ Fast and lightweight (No external libraries)

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6)
* **Storage:** Browser LocalStorage
* **Architecture:** Modular and event-driven DOM manipulation

## ⚙️ How It Works

1. User enters a task in the input field
2. Task is stored in an array and saved to LocalStorage
3. Tasks are dynamically rendered on the UI
4. Users can:

   * Toggle completion status
   * Delete tasks
   * Filter tasks based on status
5. Data persists even after page reload
